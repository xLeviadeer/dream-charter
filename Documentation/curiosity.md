# Curiosity
Uses `*` prefix. An 'unknown thing'

A curiosity is an object or potential path which is unexplored or unknown in function.

## Formatting
`* <text>`
* `text` can be any string

### Example
> ```
> * huge scary dragon
> ```

## Information Nesting
[Information](./information.md) can be nested inside of a curiosity

### Example
> ```
> * strange old door
>	= requires 1 apple to open
> ```
