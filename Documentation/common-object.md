# Common Objects
Uses `^` prefix. A 'common object reference'

## Formatting
`^ <object_name>`
* `object_name` is expected to be a valid [common object](./overview.md#common-objects) name

### Example
> ```
> ^ fruit
> ```

## Information Nsesting
[Information](./information.md) can be nested inside of a common object reference

### Example
> ```
> ^ fruit
>	= restores 10 health
> ```
