# Objects
Uses `+` prefix. An 'object'.

## Formatting
`+ <name>` 
* `name` can be any string 

### Example
> ```
> + carrot
> ```

## Information Nesting
[Information](./information.md) can be nested inside of an object

### Example
> ```
> + apple 
>	= restores 5 health
> ```
