# Creation Locations
⸉creation locations⸉ are the places where files are created when using the creation action or object merge function.

## Where to Change It
The common location creation folder and common object creation folder can be changed in `./data/common_location_creation.txt` and `./data/common_object_creation.txt` respectively. Each should be the only text in the file with no new lines.

The program expects a valid, and existing folder.

Further nesting can happen╌as, the creation location only specifies where the files are created at╌not, where files can live at .
