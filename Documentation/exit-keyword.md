# Exit Keyword
The ⸉exit keyword⸉ is the keyword associated with the comma (`,`) explore action when running the program.

## Where to Change It
The exit keyword can be changed in `./data/exit_keyword.txt`. It should be the only text in the file with no new lines.

The exit keyword ⊰can⊱ include a cost as part of it as long as all references also include this cost╌if they are expected to be referencable via the comma action.

### Example
The example⊶[exit-keyword](./Examples/exit-keyword)⊷shows the exit keyword being changed. In the example, the path ⸉die⸉ in ⸉my location⸉ will be recognized as the comma action.
