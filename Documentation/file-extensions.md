# File Extensions
File extensions for data files can be changed to fit the game or your personal preferences.

## Where to Change It
The file extensions for common locations and objects can be changed in `./data/common_location_extension.txt` & `./data/common_object_extension.txt` respectively. Each should be the only text in the file with no new lines.

The program expects text to be valid file extension characters.

The example⊶[file-extensions](./Examples/file-extensions)⊷shows the default common location extension being changed.
