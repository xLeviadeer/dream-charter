# Default Location
The ⸉default location⸉ is the starting location for which the program starts at and uses as ⸉home⸉. This should be the place you spawn when entering the game. 

## Where to Change It
The default location can be changed in `./data/default_location.txt`. It should be the only text in the file with no new lines.

The program expects of a location file of the name of the default location to exist or it will fail.

The example⊶[default-location](./Examples/default-location)⊷shows the default location being changed.
