# Overview
The Dream Charter is a program intended for taking notes while playing level based large exploration games. It is intended for games like [Yume Nikki](https://en.wikipedia.org/wiki/Yume_Nikki) but can be used for any similar games.

The Dream Charter provides a common format for specifying information fields which can be proccessed and operated on to determine complex conclusions which would be taxing to do by hand such as pathfinding, keeping track of where tasks need to be completed at different locations or finding locations with specific attributes.

**First time using the Dream Charter or need a refresher? To get started read [Getting Started](./getting-started.md).**
